개요

웹에 검색기능을 추가하기까지의 정리

현재 상황

하려는 것

위 그림에서 버튼을 누르면 입력란에 입력된 텍스트를 코드에서 가공해야하는데
입력된 텍스트를 가져오는 것에서 막힘

설명

앞으로 추가할 것

  1. 검색한 프로그램들 즐겨찾기할 수 있는 기능 추가
  2. 검색 시 한번에 프로그램을 10개씩 보여주고 더보기누르면 더 보여주도록 (최대 50개 프로그램)
  3. 즐겨찾기한 프로그램들이 최근에 언제 올라왔고 몇화까지 올라왔는지 갱신
